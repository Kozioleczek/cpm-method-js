Instrukcja obsługi
  1. W celu wyczyszczenia formularza kliknij przycisk "Reset"
  2. Aby dodać wiersz kliknij "Dodaj wiersz"
  3. Formularz nie posiada walidacji, w celu poprawnego działania dbaj o poprawność danych
    • Pierwsze zadanie powinno mieć czas trwania 0 oraz brak zalenoznosci
    • Zależności powinny być zapisane duza litera, jesli jest ich wiecej przedzielone przecinkiem
  4. Po wypełnieniu tabeli kliknij "Generuj graf"
